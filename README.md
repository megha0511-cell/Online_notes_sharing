# 📚 Online Notes Sharing System

## 📌 Project Description
Online Notes Sharing System is a web-based application developed using JSP and Servlets.  
The system allows users to upload, manage, and access study notes online in an organized manner.

This project is designed to simplify sharing academic materials between students.

---

## 🚀 Technologies Used

- Java (JDK 8+)
- JSP (Java Server Pages)
- Servlets
- HTML
- CSS
- Apache Tomcat Server
- NetBeans IDE
- MySQL (if database used)

---

## 🎯 Features

- User Registration & Login
- Upload Notes
- View / Download Notes
- Manage Notes
- Secure Authentication
- Organized File Storage

---

## 🗂️ Project Structure

- `src/` → Java Servlet Files
- `web/` → JSP Pages & Frontend
- `nbproject/` → NetBeans Configuration
- `build.xml` → Project Build File

---

## 🛠️ How to Run the Project

1. Install NetBeans IDE
2. Install Apache Tomcat Server
3. Import the project into NetBeans
4. Configure Tomcat Server
5. Run the project

---

## 🧠 Purpose of the Project

The purpose of this project is to:
- Provide a centralized platform for sharing study materials
- Improve accessibility of notes
- Learn JSP & Servlet based web development

---

## 🗄️ Database Setup

1. Open MySQL Workbench
2. Create a new database
3. Import the file `database/notes_db.sql`
4. Run the script
